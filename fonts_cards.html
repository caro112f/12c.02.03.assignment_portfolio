Assignment:
Fonts and Cards

About the project

In this project the focus was on choosing fonts and figuring out the compatability of the different font pairs. Futhermore the scaling of fonts in different heading types was calculated with the help of a type scaler.

Selected details

Choosing fonts

The fonts chosen for this assignment are Audiowide font for h1-h6 and the Raleway font for p-texts.

Displaying list of cards with grid

The layout for the UI-cards are made using grid, as this CSS-tool is helpful for making more precise layouts.

Aligning content inside card with flex

The content within the UI-cards uses flex for layout. Flex is ideal for this content as the different properties for flex, such as flex-direction, makes it easier to transition between mobile and web, and therefore ensures responsiveness.

Adding scale effect on hover

For effect the UI-cards have a scale CSS animation. When the user hovers over a card, the card scales to 1.03 of its size with an ease-out transition.
