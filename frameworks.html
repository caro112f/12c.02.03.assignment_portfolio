Assignment:
Designing with Framework

About the project

In this assignment I have used the framework Pico.css to make a responsive website easy and fast. By adding classes from the Pico website the layout changes according the CSS-styling the class adds.

Selected details

Pico class for responsive grid

By using the class .grid the layout of these components changes to match the grid. I have sectioned the content into two seperate div and used the class to make a more interesting grid layout.

Pico automatic contact form

The contact form was easily styled with the automatic css styling from the framework. The contact form fits both laptop and mobile screens.

Responsive images

The images are automatically scaled to match the parent div when using the Pico framework, therefore no extra CSS rules were added to the images.
