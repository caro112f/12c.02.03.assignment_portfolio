Assignment:
Responsive Design

About the project

This website has its focus on responsiveness using Grid and Flex. I have worked in Iphone and Ipad view in the Firefox inspector mode when making the site responsive.

Selected details

Responsive grid layout

Here is an example of two different layouts of the same content between web and mobile. The web uses a grid layout and the mobile version uses a single column layout. This way the content is readable in both formats.

Ipad screen size included

To ensure a comfortable experience for all types of users, the Ipad screen size is included in the making of the website. In some cases the layout from the laptop version fits well with the Ipad, in other cases the more simple layout of the mobile version fits better. If neither makes a good fit, the Ipad screen size gets its own layout.

List view in web and mobile

The list should be easy to read in the mobile, therefore making it the main content, and the graphical element is either placed over or under the text. In web, the screen is big enough to have some fun with the layout and therefore make the graphical element and text sit next to each other in a more enclosed layout.

Contact form in web and mobile

Another example of a specific type of content that can be changed, this time using Flex and flex-direction. The mobile version fits better with a flex-direction column, whereas the web version uses a combination of row and column.
